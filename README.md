# 🚗 AI Car Recommendation System

An intelligent full-stack web application that helps users find the most suitable car based on their preferences using AI-powered recommendations and conversational chat.

The system uses a **Spring Boot backend**, **React frontend**, and integrates with **Google Gemini AI API** to generate smart, context-aware car suggestions through chat interaction.

---

## 🌐 Live Demo

- 🔗 Frontend: https://car-dekho-samprita.netlify.app/
- 🔗 Backend: https://cardekhoassignment-production.up.railway.app/

---

## 📌 Features

- 🤖 AI-powered car recommendations using natural language input  
- 💬 Chat-based interface for user interaction  
- 🚘 Smart suggestions based on budget, fuel type, brand, and usage  
- ⚡ Real-time API communication between frontend and backend  
- 📊 Structured recommendation output  
- 📱 Fully responsive UI (mobile + desktop friendly)  
- 🔗 RESTful API integration
